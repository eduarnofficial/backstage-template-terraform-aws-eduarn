# EduArn Backstage + Terraform Demo

EduArn.com Retail and Corporate Training | Best LMS for DevOps, Cloud & Platform Engineering

This project demonstrates how to use Backstage as a Developer Portal to automate infrastructure provisioning using Terraform and GitHub.

It is designed for training sessions where freshers and professionals learn real-world Platform Engineering, DevOps automation, and Internal Developer Platform (IDP) concepts.
